«Приложение» - Приложение на Платформе CUBA.

(в) корневой пакет Приложения,

(д) Диапазон Допустимых Дат Версий.

«Бесплатный Лицензионный Файл» - Лицензионный Файл, предоставляемый Лицензиату бесплатно в составе Платформы CUBA при загрузке с сайта www.cuba-platform.ru.

«Сессия» - период активности пользователя или процесса между входом и выходом из приложения.

«Сущность Предметной Области» - категория данных, используемая в Приложении. Технически, сущность предметной области представлена аннотированным Java классом, представляющим элемент модели данных Приложения. Сущность предметной области характеризуется атрибутами и связями с другими сущностями. Она может быть сохранена в базу данных, а также отображена в интерфейсе пользователя посредством стандартных средств Платформы CUBA. Например, если Платформа CUBA используется для создания приложения для управления библиотекой, сущности предметной области могут включать в себя "книгу", "автора" или "издателя". Cущности предметной области, включенные в платформу (например, com.haulmont.cuba.security.entity.User), не учитываются с точки зрения лицензирования.

«Диапазон Допустимых Дат Версий» - диапазон дат, указанный в Лицензионном Файле, который ограничивает даты релизов Новых Версий Платформы CUBA, которые могут использоваться в Экземпляре Приложения.

«Новая Версия» - версия Платформы CUBA, которая может быть опубликована Лицензиаром на сайте www.cuba-platform.ru в любое время на его усмотрение.
